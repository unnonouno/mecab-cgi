
CGI Sample Appilication with pficommon
======================================

REQUIREMENT
-----------

- pficommon (https://github.com/pfi/pficommon)
- mecab (http://mecab.sourceforge.net/)
- python (for waf)


HOW TO BUILD
------------

::

 $ ./waf configure
 $ ./waf


HOW TO RUN
----------

::

 $ ./build/src/mecab-cgi

Access to http://localhost:8080 with your browser!
